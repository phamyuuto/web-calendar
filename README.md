# Calendar Web App

## ✨ Features

- 📅 Chế độ xem theo ngày, tháng và năm hiện tại
- 🔁 Chuyển đổi giữa ngày, tháng và năm
- 🧭 Chuyển đổi giữa chế độ xem tháng, năm và thập kỷ
- ⏱ Hẹn giờ phiên làm việc có thể tùy chỉnh (từ 5 đến 240 phút)

## 🔧 Technologies Used

- HTML & CSS & JavaScript
